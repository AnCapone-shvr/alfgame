# AlfGame 🐐

Gioco mobile stealth/puzzle antispecista. Un gruppo ALF libera gli animali da allevamenti, zoo, laboratori e circhi.

**Stato**: in sviluppo attivo — livello 1 (allevamento) giocabile.

## Come provarlo

Apri `index.html` nel browser. Nessuna installazione necessaria.

## Struttura

```
index.html          ← il gioco (file unico, autocontenuto)
design.md           ← documento tecnico / stato del progetto
CLAUDE.md           ← istruzioni per Claude Code
assets/             ← PNG sorgente degli sprite
```
